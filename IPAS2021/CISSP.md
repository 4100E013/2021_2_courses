# CISSP 資訊安全通識體系八大領域

- CISSP®(Certified Information Systems Security Professional)是國際公認最具權威的資訊安全專業人員證照。
- 如果您在資訊安全領域有多年實務工作經驗，或者您想要向其他人證明您是資訊安全領域的佼佼者，那麼CISSP®認證就是您的下一個目標。
